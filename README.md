# GestSoutenance

**Stack :** HTML · CSS · JavaScript · PHP · MySQL

## Installation

1. Copier dans `C:\xampp1\htdocs\travail-groupe-s6`
2. Démarrer Apache + MySQL (XAMPP)
3. phpMyAdmin → Importer `database/schema.sql` puis `database/seed.sql`
4. Ouvrir http://localhost/travail-groupe-s6

## Structure

```
config/              → database.php, app.php
database/            → schema.sql, seed.sql
includes/            → auth, fonctions, modèles
pages/admin/salles/  → CRUD salles
assets/              → CSS + JS
login.php            → Connexion
index.php            → Accueil
```
